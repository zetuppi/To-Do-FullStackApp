## Sobre o Projeto

Este é um projeto de aplicação web, desenvolvido no **Contexto acadêmico para a materia de projeto Fullstack**.

---

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias principais:

* **Vite** (Ferramenta de Build)
* **TypeScript** (Linguagem)
* **React** (Biblioteca de Interface)
* **shadcn-ui** (Componentes de UI)
* **Tailwind CSS** (Framework de Estilo)

---

## Como Editar e Executar o Código

Você pode trabalhar neste projeto usando sua IDE de preferência.

### Pré-requisitos

Certifique-se de ter o **Node.js** e o **npm** instalados.

### Passos de Instalação e Execução

Siga os comandos abaixo para configurar e iniciar o projeto localmente:

```sh
# Passo 1: Clone o repositório
git clone <URL_DO_SEU_REPOSITORIO>

# Passo 2: Navegue até a pasta do projeto
cd <NOME_DO_SEU_PROJETO>

# Passo 3: Instale as dependências
npm install

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
